Последовательность решения задач Микросервисов и node.js
1) Клонируем репозиторий github, указанный в задании
2) Заходим на www.glitch.com, регистрируемся через github
   Выбираем "Создать сайт", выбираем Обычный (не Node.js!) и "Клонировать репозиторий", 
   вставляем ссылку репозитория git -  https://github.com/LLPeterX/camp.github.io.git
   (Если был выбран "обычный сайт", то потом можно поменять)
4) После редактирования файла (package.json) жмем кнопку "Share",
   копируем URL из поля "Live Site" и вставляем в поле "Solution Link" на freecodecamp.org
   URL: https://spotless-hollow-risk.glitch.me
   Это надо делать после каждого изменения на glitch.
   И дождаться когда App Status станет OK.
5) Не забываем push измененных локальных файлов на github.com

Версии NPM:
 ~x.y.x - использовать последнюю версию (x - major) пакета
 ^x.y.z - использовать последнюю версию minor (y) пакета
